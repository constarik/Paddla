# PADDLA

Provably Fair интерактивная аркадная игра с управляемым бампером.

## Концепция

Эволюция Ball Rush с добавлением:
- **Интерактивность** — игрок управляет бампером (круг, 2× размер шара)
- **Provably Fair** — VRF-верификация честности
- **Детерминированная физика** — replay любой игры

## Структура

```
PADDLA/
├── engine/          # Игровой движок (физика, RNG, логика)
├── simulation/      # Тесты RTP, анализ баланса
├── server/          # VRF, API, протокол
└── client/          # UI, рендеринг
```

## Механика Бампера

- Радиус: 0.4 (шар: 0.2)
- Зона: y ∈ [5.3, 8.6] (нижняя часть поля)
- Управление: follow cursor/touch
- Скорость: max 0.15 за тик (3× скорость шара)

## Provably Fair

```
1. Игрок → client_seed
2. Сервер → VRF(secret_key, client_seed + game_id) → (game_seed, proof)
3. Игра с записью input_log (позиции бампера)
4. Игрок может верифицировать: replay(game_seed, input_log) = результат
```

## Разработка

```bash
npm install
npm run sim      # запуск симуляции
npm run dev      # локальный сервер
```
